# Gaurish Budhiraja — Portfolio Website

A modern  portfolio built to showcase projects, technical skills, certifications, and engineering-focused work across cybersecurity, DevSecOps, Linux security, and software development.

The portfolio highlights hands-on projects involving intrusion detection systems, Linux-based security tooling, endpoint telemetry, full-stack development, and network security experimentation.

---

## 🚀 Features

- Modern responsive portfolio interface
- Cybersecurity and systems engineering focused design
- Featured project showcase
- Technical skills and certifications section
- GitHub and LinkedIn integration
- Responsive UI for desktop and mobile
- Smooth animations and interactive components
- Fast deployment using Vercel

---

### UI / Styling
